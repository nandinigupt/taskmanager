Task Management Web Application (MERN Stack)
Overview

This project is a full-stack Task Management Web Application built using the MERN stack.
It allows users to create, view, update, and delete tasks with persistent storage and a responsive user interface.

Features

Create, read, update, and delete tasks

Task fields: Title, Description, Status

Responsive and professional UI

RESTful API architecture

Persistent data storage using MongoDB
